# **App Name**: ReelGenius

## Core Features:

- Reel Idea Input: Allows users to input a brief idea for a reel.
- Hook Generation: Generates 5 catchy hooks based on the user's reel idea, leveraging AI to maximize audience engagement, using its access to various tools to get it done.
- Detailed Description Generation: For each hook, generates a detailed description on how to shoot and edit the reel, including transitions and highlighted words. The suggestions are output in a bulleted format, using AI, leveraging different tools to find creative options.
- Content Highlighting: Highlights key words and phrases within the descriptions to emphasize important aspects of the shooting and editing process.
- Description Formatting: Presents the generated descriptions in a well-organized, bulleted list format for easy understanding and execution.

## Style Guidelines:

- Primary color: Deep teal (#4DB6AC) to evoke creativity and trustworthiness.
- Background color: Light beige (#F5F5DC) to create a neutral and calm canvas.
- Accent color: Soft coral (#F08080) for highlighting important elements and CTAs.
- Headline font: 'Poppins' (sans-serif) for a modern, clean, and fashionable look for headings.
- Body font: 'PT Sans' (sans-serif) which will provide the ideal pairing to the modern headline typeface
- Use clean, minimalistic icons related to video creation, editing, and social media sharing.
- Clean and structured layout to guide users through the process of inputting ideas and reviewing generated content. The suggested formats make it simple to extract relevant info at a glance.
- Subtle transitions and animations to provide visual feedback and engagement without being distracting.
